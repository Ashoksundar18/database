# Neon Database Example

This repository contains example code for connecting to a [Neon](https://neon.tech/) serverless Postgres database using Node.js.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Copy the `.env.example` file to `.env` and replace the placeholder with your actual Neon database connection string:
   ```bash
   cp .env.example .env
   ```

3. Run the code:
   ```bash
   npm start
   ```
